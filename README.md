# Enter the Arena V8

A self-hosted Pokémon Champions team-preview assistant.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS v4 foundation
- Framer Motion
- TanStack Query
- Dexie / IndexedDB
- Progressive Web App
- Nginx
- Docker Buildx
- GitHub Container Registry

See `README-GHCR.md` for repository, registry, Unraid, and Docker Compose instructions.
